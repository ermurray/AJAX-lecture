## Pros/cons of SPA
- Pros
  - can be a better UX. more features faster and reactive without leaving the page 
  - seamless interface
- Cons 
  - initial loads "can" be worse, lots of javascript
  - loose browser functionality ( back, forward, refresh)
  - SEO can be worse
